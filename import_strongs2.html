Strong's Mapping Import v2

Uses the kennethreitz/kjvstudy.org interlinear dataset (~12MB gzipped). Maps each Hebrew/Greek word to its single primary content word in the KJV (skipping leading articles/prepositions in multi-word glosses, e.g. "In the beginning" → "beginning"). Clears and replaces existing strongs_mapping data.

Requires modern browser (Chrome 80+ / Safari 16.4+) for gzip decompression. Keep this tab open until complete.
